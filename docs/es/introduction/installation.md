# Instalación

Nix puede instalarse de varias maneras:

1. Como gestor de paquetes en macOS, Linux o WSL.
2. Como gestor del entorno del sistema en NixOS, una distribución de Linux que utiliza Nix
   para la gestión del sistema.

Este libro se centra principalmente en el uso de NixOS y Flakes. Por lo tanto, omitiremos
el contenido que corresponde únicamente a Nix (como la instalación en macOS, Linux o WSL).

El proceso de instalación de NixOS es sencillo, pero no profundizaremos en los detalles
aquí. Para más información, visita el sitio oficial de descargas en
<https://nixos.org/download.html>.

> Si usas macOS,
> [ryan4yin/nix-darwin-kickstarter](https://github.com/ryan4yin/nix-darwin-kickstarter)
> puede ser un buen punto de partida para ti. Puedes aprender a usar Nix con este libro y
> tomar _nix-darwin-kickstarter_ como base para construir tu propia configuración de Nix.
